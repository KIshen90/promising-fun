### Prerequisites

You need to have Node installed. Visit https://nodejs.org/en/ to download and install.

### Install Mocha

Run the following:

```
npm install
npm install mocha -g
```

### Run Tests

Changes to code in the following will rerun the tests automatically.

1. Basic tests:

```
npm run test1
```

2. Next:

```
npm run test2
```
